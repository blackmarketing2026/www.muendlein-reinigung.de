# Fonts

Schriftarten für das Projekt.

Aktuell wird Inter über Google Fonts geladen (in includes/head.shtml).

Wenn Self-Hosting der Schrift gewünscht wird, Dateien hier ablegen:

- inter-regular.woff2
- inter-medium.woff2
- inter-semibold.woff2
- inter-bold.woff2
- inter-extrabold.woff2

Und @font-face in style.css eintragen.
